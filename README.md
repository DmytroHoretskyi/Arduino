 Щоб запустити код вам потрібно в папці backend вписати команджу "npm i" для встановлення залежностей, після цього можна запускати backend частину за допомогою команди "npm start".

Код для ардуіно запускається через кманду "npm ./app.js"

Також для запису данних в базу вам потрібно мати локально встановлений PostgreSQL і створити там базу з назвою "arduino", або ж вписати назву вашої бази данних, логін і пароль в файлику db.js

frontend частина: для початку потрібно також написати "npm i" для встановення залежностей
після цього можна запускати фронт командою npm start
